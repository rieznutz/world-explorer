# World Explorer (OOP-style)

Electron desktop app that searches countries (REST Countries API) and lets you save itineraries (localStorage). This version uses OOP classes for clarity.

## Run
1. npm install
2. npm start

## Features
- Search countries many times (no single-use bug)
- View details (flag, capital, region, area, languages, timezones)
- Save search results to itinerary
- Full CRUD for itineraries (add, edit, delete, clear)
- Manual add for custom plans
